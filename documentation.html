aes_encryption.py

encrypt(self, plain_text, key, nonce)

Encrypts the given plain_text using AES in CTR mode with a 256-bit key.

Args:
Plain_text(str): The message to be encrypted
Key(aes_key): The generated AES key used for encryption
Nonce: The generated arbitrary number that can be used just once in the cryptographic communication.

Returns:
The encrypted message as a Base64-encoded string.

decrypt(self, ciphertext, key, nonce)

Decrypts the given ciphertext using AES in CTR mode with a 256-bit key.

Args:
Ciphertext: The encrypted string which is to be decrypted.
Key(aes_key): The generated AES key used for encryption
Nonce: The generated arbitrary number that can be used just once in the cryptographic communication.

Returns:
The decrypted message as a bytes object.

rsa_encryption.py

generate_key_pair(self, b=2048)

Generate a pair of public and private keys for RSA encryption.

Args:
does not receive any explicit arguments.
b: The default bit size used is 2048

Returns:
A tuple containing the public key and the private key.

encrypt(self, message, public_key)

Encrypt a message using RSA encryption with a public key.

Args:
  message (str): The message to be encrypted.
  public_key (rsa.PublicKey): The public key to be used for encryption.

Returns:
The encrypted message as a Base64-encoded string.

decrypt(self, encrypted_message, private_key)

Decrypt a message using RSA encryption with a private key.

Args:

  encrypted_message (str): The encrypted message as a Base64-encoded string.
  private_key (rsa.PrivateKey): The private key to be used for decryption.
Returns:

  The signature as a Base64-encoded string.

sign(self, message, private_key, sign='SHA-256')

Sign a message using RSA encryption with a private key.

Args:
  message (str): The message to be signed.
  private_key (rsa.PrivateKey): The private key to be used for signing.
  sign (str): default rsa sign used is SHA-256

Returns:
The signature as a Base64-encoded string.

verify(self, message, signature, public_key)

Verify a message's signature using RSA encryption with a public key.

Args:
  message (str): The message whose signature is to be verified.
  signature (str): The signature to be verified as a Base64-encoded string.
  public_key (rsa.PublicKey): The public key to be used for verification.

Returns:
True if the signature is valid, False otherwise.

aes_encrypt(self, clear_text, key_base64, nonce_bytes=16, from_base64=True, encoded_text=False)

Args:

from_base64(bool): True if the key provided is base 64 encoded
encoded_text(bool): True if the clear_text provided is base 64 encoded

Returns:

Encryption.py

abstractmethod encrypt()

Encrypts the given clear text string according to the algorithm implementation in child class and returns the encrypted text.

abstractmethod decrypt()

Decrypts the given encrypted string according to the algorithm implementation in child class and returns the clear text.

Easy to setup

1) Clone the git repository

2) Python version >= 3.9

3) Pip install the following:
    i. cryptography
    ii. pycryptodome
    iii. rsa
    iv. base64
    v. secrets
    vi. hashlib
    vii. Crypto.Cipher

4) Directly use the util ( You can also use the (xyz)_encryption.py classes directly to make your own new util -- See usage in the comments in the code)

Note: To understand the flow you can also refer to tests_proj_4a/driver.py

Voila!
